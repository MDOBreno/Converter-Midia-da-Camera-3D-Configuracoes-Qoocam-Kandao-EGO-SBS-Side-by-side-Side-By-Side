Converting EGO videos and photos to spatial videos and photos. macOS 14 or above is needed.

Steps:
1. Put mp4 or jpg files to be converted inside a folder;
2. Run the Application and click the "select a folder" button, choose the folder mentioned in the step 1;
2. Click the "convert" button and the application will start to convert the files in the selected folder;
3. After the tasks finish, the converted files will be saved in the "output" folder and it will open automatically.


用于将QooCam EGO的视频和照片转换为空间视频和照片格式。
需要 macOS 14 或更高版本。

步骤：
1、把要转换格式的mp4或jpg文件放进一个文件夹中；
2、运行应用，点击“select a folder”按钮，然后选择第一步提及的文件夹；
3、点击“convert”按钮即开始对被选中的文件夹内的文件转换格式；
3、当所有任务完成后，转换格式后的文件会保存在“output”文件夹，并自动打开。
